## 커밋 메시지

[Conventional Commits](https://www.conventionalcommits.org/) 형식 사용:

```
<type>(<scope>): <subject>
```

| type | 용도 |
|------|------|
| `feat` | 새 기능 |
| `fix` | 버그 수정 |
| `docs` | 문서만 |
| `refactor` | 리팩터링 (기능 변화 없음) |
| `test` | 테스트 추가/수정 |
| `chore` | 빌드·설정 변경 |

body/footer: BREAKING CHANGE 시 필수, 그 외 선택.
